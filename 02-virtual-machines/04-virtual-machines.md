# Virtual Machines

## What is a Server?

A server is a system (physical or cloud-based) that provides services, resources, or data to other computers over a network.

---

## Virtualization

Virtualization is the process of creating multiple isolated environments from a single physical machine.

It allows one physical server to be divided into multiple virtual systems, improving resource utilization and efficiency.

---

## Virtual Machine (VM)

A Virtual Machine is a software-based computer that runs inside a physical machine.

It behaves like a real computer with its own:

- Operating system
- CPU
- Memory
- Storage

However, it shares the physical hardware of the host machine.

---

## Hypervisor

A Hypervisor is software that creates and manages Virtual Machines.

It controls how resources (CPU, memory, storage) are allocated to each VM.

There are two types:

- Type 1: Runs directly on hardware (bare metal)
- Type 2: Runs on top of an operating system
